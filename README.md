# Multiplayer Snake Game

## Overview

This is a multiplayer snake game server implemented in Python

## Usage

- The preferred usage is to install `uv` and run the server with `uv run src/main.py`.
- Instructions to install `uv` can be found [here](https://pypi.org/project/uv/).
- You can also use PyCharm and import the project as a `uv` project, then run the `main.py` file.

## Supported Python Versions
- Python=3.14
